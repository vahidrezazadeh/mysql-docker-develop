
## Run Locally

Clone the project

```bash
  git clone https://github.com/vahidrezazadeh/mysql-docker-develop.git
```

Go to the project directory

```bash
  cd mysql-docker-develop
```

Run With Docker

```bash
  docker-compose up -d
```
